## 📖 简介

本仓库收录了一套专为**学术科研场景**精心设计的 AI 提示词（Prompt）合集，覆盖论文写作的全流程需求。无论你是需要将中文稿件翻译为高质量英文、检索真实权威的学术文献、诊断实验数据中的异常规律，还是对论文段落进行发表级润色，这里都有对应的工具提示词。

## 🚀 快速开始

1. 在上方目录中选择你需要的提示词
2. 点击对应文件链接，复制提示词全文
3. 粘贴至你使用的AI平台（Claude / ChatGPT等）的对话框最前端
4. 按照提示词中的初始化话术开始使用

## ⚠️ 使用须知

- 本提示词集旨在**辅助学术写作**，所有最终内容仍需作者本人核查与负责
